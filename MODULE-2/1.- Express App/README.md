# Running the example

## You must have Node.js install in your computer

```node
https://nodejs.org/en/download/
```

## Install Dependencies

```node
npm install
```

## Run App

```node
node .
```

## Routes

We have two different available routes /data and /users

```node
127.0.0.1/data
127.0.0.1/users
```

Check them out!
